Unified Platform for IT Departments - Engineering Colleges

🎯 Core Purpose

A centralized platform to streamline academic content, placement preparation, community engagement, and student-faculty-TPO interaction for engineering IT departments.

🔐 Key Features by Role

👨‍🎓 Student

  • View/download resources
  • Subject materials & quizzes
  • Coding practice & contests
  • Placement preparation tests
  • Request mock interviews
  • Join community discussions

👩‍🏫 Faculty

  • Upload course materials
  • Create quizzes
  • Monitor student progress
  • Conduct mock interviews
  • Answer forum questions
  • Post announcements

🏢 TPO

  • Manage company profiles
  • Schedule placement drives
  • Create placement tests
  • View performance analytics

🧑‍💼 Alumni

  • Update work experience
  • Share company reviews
  • Offer mentorship
  • Share job referrals
  • Post industry updates

👨‍💻 Admin

  • User management
  • Role assignment
  • Content moderation
  • System monitoring
  • Analytics dashboard
🛠️ Technical Implementation
  • Frontend: React + TypeScript + Shadcn UI
  • Backend: Express.js with TypeScript
  • Database: PostgreSQL + Drizzle ORM
  • Authentication: JWT with Role-based Access
  • File Storage: Enabled for uploads
